本文介绍了我在工作中会用到哪些软件。

1. BCompare

极其强大易用的内容对比软件，当然，我是用的破解版。同时用IDEA内置compare with也解决了大部分code diff and merge。

2. [开源] Find And Run Robot(FARR)

Find and Run Robot一款可快速搜索电脑中文件的免费工具。 Keypirinha 也不错，但由于已经养成习惯， 仍然坚持使用FARR。

下载地址：http://www.donationcoder.com/forum/?board=12.0

3. [开源] SwitchHosts

SwitchHosts! 是一个管理、切换多个 hosts 方案的工具。它是一个免费开源软件。

下载地址：https://github.com/oldj/SwitchHosts/releases

4. BareTail Pro && BareGrep Pro

Windows下超强日志工具BareTai, 实时监控查看日志必备。

5. Cmder

漂亮的控制台模拟器，替换了windows默认丑陋的cmd控制台。 同时也在用 Hyper

下载地址: https://cmder.net/

6. [开源] VSCode

微软大名鼎鼎的编辑器。不再使用 Atom。

下载地址: https://code.visualstudio.com/

7. [开源] Ditto

一款强大的 Windows 剪贴板增强工具。

下载地址: https://ditto-cp.sourceforge.io/

8. [开源] DBeaver

多数据库管理工具，用来连MySQL。不再使用 Navicat。

下载地址: https://dbeaver.io/

9. Everything

基于名称快速定位文件和文件夹,快速查找文件必备。

下载地址: https://www.voidtools.com/sssmall2.gif

10. eDiary(电子日记本)

主要看中有加密和日历功能，方便存一些私密数据。

下载地址: http://www.haoxg.net/

11. FoxitReader.exe

查看PDF文档，看中了它的文件打开速度，快。

下载地址: https://www.foxitsoftware.cn/

12. [开源] Git

吃饭必备。

下载地址：https://git-scm.com/

13. geek

一款强制卸载清除工具,可以标准删除程序在你电脑上留下任何垃圾文件。有时候装了一些垃圾软件，用这个清理一下。

下载地址: https://geekuninstaller.com/

14. [开源] Hyper

一款 JS / HTML/ CSS 终端工具。旨在为用户创建一个美观的、易于扩展的命令行接口工具。

下载地址: https://github.com/zeit/hyper

15. Intelij IDEA

吃饭必备，唯一掏真金白银购买使用的软件。从此Eclipse相忘于江湖。另外使用 PyCharm, WebStorm, 由于IDEA内置了数据库连接组件，现在 DBeaver 和 DataGrip 偶尔使用。

下载地址: https://www.jetbrains.com/idea/

16. jd-gui

小巧的java反编译工具，同样IDEA已经有反编译插件可用，现在也很少使用了。

下载地址: http://java-decompiler.github.io/

17. Kitematic

一款跨平台可视化 Docker 容器管理工具, 但经常使用命令或脚本方式管理Docker, 偶尔使用。

下载地址: https://kitematic.com/

18. Licecap

录制屏幕操作，文件小巧易用。提PR，和测试交流必用。

下载地址: https://licecap.en.softonic.com/

另外一款在线的录屏网站也很好用 ：recordscreen.io

19. [开源] Notepad++

使用已很多年，到现在仍然好用，打开速度极快，多用 查找/替换 功能，使用正则表达式搜索非常好用。

下载地址: https://notepad-plus-plus.org/

20. Process Explorer

查杀windows进程，好用~

下载地址: https://docs.microsoft.com/en-us/sysinternals/downloads/process-explorer

21. PyCharm

写 Python 代码，Project级别的上此牛刀，小的直接用 VSCode.

下载地址: https://www.jetbrains.com/pycharm

22. Robo 3T

MongoDB 可视化管理工具

下载地址: https://robomongo.org/

23. RDM(Redis Desktop Manager)

Redis 管理工具

下载地址: https://redisdesktop.com/

24. Snipaste

截图神器

下载地址: https://www.snipaste.com/

25. SpaceSniffer

可视化的查看目录大小占用情况

下载地址: https://spacesniffer.en.softonic.com/

26. Typora

写Markdown软件，现在更习惯直接在IDEA里面写，配git版本管理MD文件。

下载地址: https://www.typora.io/

27. WinSCP

FTP/SFTP 文件传输管理软件，可惜了Xftp/Xshell系列太贵，破解版也经常过段时间失效。多用IDEA自带 SSH/FTP/SFTP功能，没开IDEA时辅助使用。

下载地址: https://winscp.net/eng/docs/lang:chs

28. XMind

画思维导图/头脑风暴

下载地址: https://www.xmind.cn/

29. Xampp

本地快速搭建一套 Apache+MySQL+PHP+PERL 环境。 以前玩wordpress使用，写一写PHP。现在多用Docker下的mysql, PHP也.

下载地址: https://www.apachefriends.org/index.html

30. Docker

到现在如果开发还没有用到Docker，就out了。

下载地址: https://www.docker.com/
